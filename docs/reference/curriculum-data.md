(EN842017), Khon Kaen University Faculty of Engineering, Digital Media Engineering.

("Course Details" / "Study Plan" pages of the prototype, Sprint 1) and the 4-category
elective grouping on report pages 25-27 (Sprint 2 increment).

## Course type / category taxonomy

The prototype's course-filter UI groups every course under one **Course Type**:

- `General` — General Education courses
- `Major (Basic)` — Major Basic courses
- `Major (Core)` — Major Core courses
- `Major (Elective)` — Major Elective courses, further split into 4 **Category**
  tracks (this is the classification finalized for DME Explorer, matching the real
  KKU DME major-elective tracks):
  - **AI** (`AI & Data` in the source filter UI)
  - **Digital Media** (`3D & Animation` in the source filter UI — labeled
    "Digital Media Industrial Courses" on the course-list page)
  - **Interactive** (`Game Dev` in the source filter UI — labeled "Interactive
    Digital Media Courses" on the course-list page)
  - **Software** (`Software` in the source filter UI — labeled "Software
    Development Course" on the course-list page)
- `Field` — Field Experience courses
